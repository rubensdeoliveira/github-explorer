Esta aplicação foi criada com o intuito de explorar as funcionalidades da API do Github.

### Tecnologias

- Typescript
- ReactJS
- React Hooks
- Eslint + Prettier + EditorConfig
- Styled Components
- react-router-dom
- react-icons
- polished
